You can point out that columns are used for **cooling**

For the header, you must add the "**isCooling**" field (boolean)

The header **automatically groups the cooling columns**, and adds the text "**Cooling**" above

```json
{
    isCooling: true,
    label: "Header 3"
}
```

And for the content you must add the "**hasCooling**" field (boolean) for each line

```json
{
    hasCooling: true,
    label: "Row 1/3"
}
```

For example, if you want columns **3, 4 and 5** to represent cooling values

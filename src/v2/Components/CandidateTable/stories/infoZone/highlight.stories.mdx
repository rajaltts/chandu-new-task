To **highlight** certain results, you can add the "**isHighlighted**" field for the lines concerned

This can be useful to highlight the results of the calculation engine for example

Like this:

```json
"content": [
    {
      "id": 0,
      "rowContent": [
        {
          "isInfoZone": true
        },
        {
          "label": "Row 1/1"
        },
        {
          "label": "Row 1/2"
        }
      ],
      "isHighlighted": true
    }
    ...
]
```

export const highlightInfoZone = createBasicConfig(3, 2, {
    highlighted: [1],
    infoZone: [0]
})

</details>

Of course you can combine this field with "isSelectable" and "selected" fields

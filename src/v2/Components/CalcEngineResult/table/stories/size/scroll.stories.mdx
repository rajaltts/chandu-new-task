If you limit the size of the parent container of your table, then it will automatically scroll

With something like that

```jsx
<div style={{ border: '1px solid black', height: '400px' }}>
    <CandidateTable ... />
</div>
```

When your table scrolls, **the header remains fixed**, only the **content lines will scroll**

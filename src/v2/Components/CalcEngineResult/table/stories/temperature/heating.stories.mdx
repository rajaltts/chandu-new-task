You can point out that columns are used for **heating**

For the header, you must add the "**isHeating**" field (boolean)

The header **automatically groups the heating columns**, and adds the text "**Heating**" above

```json
{
    isHeating: true,
    label: "Header 6"
}
```

And for the content you must add the "**hasHeating**" field (boolean) for each line

```json
{
    hasHeating: true,
    label: "Row 1/6"
}
```

For example, if you want columns **6, 7 and 8** to represent heating values
